MeowPixel MVP 设计方案

治愈系复古像素猫咪小程序 - 完整设计规格

📋 项目概述

项目名称MeowPixel
技术栈uni-app + Vue 3 + 微信小程序
开发阶段MVP + AI 聊天（Q1 + Q2）
核心模块首页、喵言喵语翻译器、喵喵杂货铺
视觉风格英短蓝猫像素风
导航结构单页 + 浮动入口

🏠 首页：猫咪小窝

核心功能

  • 展示猫咪状态与动画
  • 日夜场景自动切换（6:00-18:00）
  • 长按互动触发爱心粒子
  • 状态值实时显示

猫咪状态

  • 心情值 (0-100)：互动提升
  • 饱食度 (0-100)：喂食提升
  • 亲密度 (累计)：长期积累

浮动入口

  • 💬 喵言喵语（聊天）
  • 🛒 喵喵杂货铺（商店）
  • ⚙️ 设置（可选）

💬 喵言喵语翻译器

交互流程

  1. 用户输入文字或语音
  2. 调用通义千问 API 生成回复
  3. 猫咪以"喵语"显示（喵喵～）
  4. 点击【翻译】显示真实含义
  5. 使用复古 RPG 对话框样式

技术要点

  • 语音转文字：微信小程序 API
  • AI 对话：通义千问 API
  • 喵语生成：替换关键词为"喵"
  • 对话历史：本地存储

  • 金枪鱼罐头：+40 心情+饱食

  • 红领结套装：优雅绅士
  • 小雨衣套装：雨天可爱

  • 暖炉：解锁烤火动作

  • 小秋千：解锁荡秋千动作

💰 货币系统

获取方式

  • 互动积累：每次互动 +1~3 金币，每日上限 20
  • 签到奖励：每日签到 +10 金币，连续 7 天 +50
  • 小任务：喂食猫咪 +5，聊天 3 次 +10 等

消费场景

  • 治愈食物：10~50 金币
  • 装扮套装：100~300 金币
  • 温馨家具：200~500 金币
