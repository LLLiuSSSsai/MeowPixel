✅ 正确的像素风格方向

应该这样

  • 使用 SVG 像素图形，每个像素用 rect 绘制
  • 图标采用 8x8 / 16x16 像素网格
  • 配色使用 复古游戏调色板（有限色数）
  • 对话框使用 像素边框 + 尖角指示
  • 按钮采用 像素阴影 + 立体感
  • 动画使用 逐帧切换，非平滑过渡

避免这样

  • ❌ 使用 emoji 作为图标（🐱💬🛒）
  • ❌ 圆角矩形按钮（border-radius）
  • ❌ 渐变色背景（非像素化渐变）
  • ❌ 平滑 CSS 动画过渡
  • ❌ 现代扁平化设计风格
  • ❌ 高饱和度现代配色
